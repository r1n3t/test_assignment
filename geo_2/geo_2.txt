ArcGIS Pro: 
1. Импорт точек в ArcGIS Pro. Для этого необходимо сохранить исходный файл в формате csv.
2. Визуализация точек на карте (в ПО ArcGIS). Для этого необходимо нажать ПКМ на слой с таблицей в разделе «Содержание» и выбрать инструмент «Отобразить данные XY». Или использовать инструмент геообработки «XY таблица в точки».
3. Добавляем на карту shapefile с агломерациями 
4. Нажимаем ПКМ на слой с точками (полученный на 2 шаге), далее «Добавить соединение» -> «Добавить пространственное соединение»
5. Заполняем параметры: Целевые объекты -> точки; Присоединить объекты -> агломерации; Операция соединения -> Один к одному; Опция сопоставления -> Ближайшее; Радиус поиска -> N км; Поля -> Выходные поля -> Атрибут с названием агломерации 
6. Выгружаем таблицу с помощью инструмента «Таблица в Excel». В параметрах выбираем слой с точками и пишем путь к сохраняем данным 

QGIS:
1.	Импорт точек в QGIS. Для этого необходимо сохранить исходный файл в формате csv. Далее необходимо использовать инструмент «Добавить слой» -> «Добавить слой с разделителями».
2.	Добавляем shapefile с агломерациями. «Добавить слой» -> «Добавить векторный слой»
3.	Строим буфер вокруг точек. Для этого используем инструмент «Буфер»: «Вектор» -> «Геообработка» -> «Буфер». Указываем необходимое расстояние (N км)
4.	Делаем выборку с помощью инструмента «Выбрать по расположению». Выбираем Буферы, которые пересекаются с полигонами. Далее работаем только с данной выборкой.
5.	Далее для нахождения ближайшей агломерации используем модуль NNJoin. В параметрах Input vector layer -> Слой с Буфером, Join vector layer -> Слой с агломерациями
6.	С помощью инструмента «Экспорт в электронную таблицу» сохраняем информацию о ближайшей агломерации к точке 5Post.
